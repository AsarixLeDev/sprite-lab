# Vision providers

A vision provider is an image description service. Sprite Lab can use one to suggest short labels for your images.

Available choices may include:

- **Local:** runs on your computer and does not send images to a remote service.
- **Hosted:** sends images to the named service and may create a charge.
- **Manual:** you write or choose descriptions yourself.

If no option is set up, choose **Choose a description option**. You can connect a local or hosted option, or continue with manual descriptions.

If sign-in fails, choose **Open connection settings**, sign in again, then choose **Test connection**. Sprite Lab does not store sign-in secrets in the project.

If the connection drops, completed work is preserved and no new remote work starts automatically.
