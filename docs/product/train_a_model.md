# Train a model

Training becomes available after the dataset and required image descriptions are ready.

Before starting, Sprite Lab shows:

- where training will run;
- an estimated duration;
- expected computer use;
- any hosted spending limit and cost estimate.

Choose **Start training** once. This is the only confirmation for the complete local training operation.

For hosted training, check the named service, limit, and estimate, then choose **Confirm and start hosted training** once. Nothing starts before that confirmation.

If training is unavailable, choose **Show what to fix**. Sprite Lab protects the dataset and completed work while required safety checks are repaired.

If a run is interrupted, choose **Check saved work**. Sprite Lab verifies that the saved work still fits the current project before showing **Continue run**.
