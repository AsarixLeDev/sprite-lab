===== top-level =====
usage: python -m spritelab.dataset_v5.cli [-h]
                                          {build,verify,build-policy-preview,verify-policy-preview,compare-policy-previews,validate-contract,build-view,verify-view,freeze-view,verify-freeze} ...

positional arguments:
  {build,verify,build-policy-preview,verify-policy-preview,compare-policy-previews,validate-contract,build-view,verify-view,freeze-view,verify-freeze}
    validate-contract   Validate the Dataset-v5 named-view contract.
    build-view          Build a deterministic non-production Dataset-v5 named
                        view.
    verify-view         Verify a Dataset-v5 named view without modifying it.
    freeze-view         Freeze a complete, verified Dataset-v5 named view.
    verify-freeze       Verify a frozen Dataset-v5 named view.

options:
  -h, --help            show this help message and exit
